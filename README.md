# Event Ticketing System 🎟

## Features
- Create events with organizers and venues
- Buy tickets
- Prevent overbooking
- View tickets sold and remaining capacity

## Tech Stack
- Frontend: React (Vite)
- Backend: Node.js, Express
- Database: PostgreSQL
- ORM: Sequelize

## How to Run

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
